# Common CSS Styling Classes

## 1. Text Alignment Classes

Used to align text.

### CSS

```css
.text-center {
    text-align: center;
}

.text-right {
    text-align: right;
}
```

### HTML

```html
<h1 class="text-center">Welcome</h1>
```

---

# 2. Text Color Classes

Used to apply text colors.

### CSS

```css
.text-primary {
    color: blue;
}

.text-danger {
    color: red;
}
```

### HTML

```html
<p class="text-danger">Error Message</p>
```

---

# 3. Background Classes

Used to set background colors.

### CSS

```css
.bg-dark {
    background-color: black;
    color: white;
}
```

### HTML

```html
<div class="bg-dark">
    Dark Background
</div>
```

---

# 4. Margin Classes

Used to create outer spacing.

### CSS

```css
.mt-1 {
    margin-top: 10px;
}
```

### HTML

```html
<div class="mt-1">Content</div>
```

---

# 5. Padding Classes

Used to create inner spacing.

### CSS

```css
.p-1 {
    padding: 10px;
}
```

### HTML

```html
<div class="p-1">Box Content</div>
```

---

# 6. Border Classes

Used for borders and rounded corners.

### CSS

```css
.border {
    border: 1px solid black;
}

.rounded {
    border-radius: 8px;
}
```

---

# 7. Button Classes

Used for reusable button styles.

### CSS

```css
.btn {
    padding: 10px 20px;
}

.btn-primary {
    background-color: blue;
    color: white;
}
```

### HTML

```html
<button class="btn btn-primary">
    Submit
</button>
```

---

# 8. Shadow Classes

Used to add depth using shadows.

### CSS

```css
.shadow {
    box-shadow: 0 2px 5px gray;
}
```

---

# Advantages of Styling Classes

- Reusable styles
- Faster development
- Cleaner code
- Consistent UI design
